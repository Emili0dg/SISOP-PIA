Fundamental 1

Este trabajo explora los sistemas operativos, explicando su función como intermediarios entre el usuario y el hardware. Analiza los componentes clave de los sistemas operativos, como el núcleo, la gestión de memoria, el sistema de archivos y la seguridad.

Fundamental 2

Se centra en la multitarea y el control de concurrencia dentro de los sistemas operativos. Discute la importancia de gestionar múltiples procesos simultáneamente, las técnicas para evitar conflictos y los tipos de sistemas que permiten la multitarea, como Windows y Linux.

Fundamental 3

Este trabajo aborda la administración de memoria y sistemas de archivos en dispositivos de diferentes plataformas, desde computadoras hasta móviles. Explica los tipos de memoria (RAM, ROM, caché) y cómo cada sistema operativo maneja el almacenamiento y organiza los archivos.

Fundamental 4

Analiza la seguridad informática, describiendo distintos tipos de malware, virus y métodos de intrusión. También aborda las medidas de autenticación y seguridad a diferentes niveles, desde el usuario individual hasta redes empresariales.
